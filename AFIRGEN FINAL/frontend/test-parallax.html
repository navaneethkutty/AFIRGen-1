Parallax Features

  • 3 depth layers with different scroll speeds
  • GPU-accelerated transforms for 60fps
  • Decorative shapes with gradients
  • Respects prefers-reduced-motion
  • Disabled on mobile for performance
  • Optimized with requestAnimationFrame
Scroll Down

Layer 1: Slowest

The first parallax layer moves at 0.1x scroll speed, creating the deepest depth effect. It contains 3 large decorative shapes with low opacity for a subtle background effect.

Layer 2: Medium

The second layer moves at 0.25x scroll speed, providing a middle ground between the background and foreground. It has 5 medium-sized shapes with slightly higher opacity.

Layer 3: Fastest

The third layer moves at 0.4x scroll speed, creating the most noticeable parallax effect. It contains 8 smaller shapes that move more quickly as you scroll.

Performance Optimizations

All parallax layers use CSS transforms (translate3d) for GPU acceleration, ensuring smooth 60fps scrolling. The effect uses requestAnimationFrame to synchronize updates with the browser's repaint cycle.

Technical Details

  • Transform: translate3d(0, offset, 0)
  • will-change: transform
  • backface-visibility: hidden
  • Passive scroll listeners
  • requestAnimationFrame throttling

Accessibility

The parallax effect respects the prefers-reduced-motion media query. Users who have enabled reduced motion in their system settings will not see any parallax animations, ensuring a comfortable experience for users with vestibular disorders or motion sensitivity.
