Resources

                

                    Technologies, tools, and resources utilized in the development of AFIRGen.
                

            


            

                
                

                    

                        
                        Database Used
                    

                    

                        

                            
                            MongoDB - NoSQL database for storing complaint data and FIR records
                        

                        

                            
                            PostgreSQL - Relational database for structured legal document storage
                        

                    

                


                
                

                    

                        
                        Tools Used
                    

                    

                        

                            
                            Python - Core programming language for AI model development
                        

                        

                            
                            TensorFlow - Machine learning framework for model training
                        

                        

                            
                            React.js - Frontend framework for user interface development
                        

                        

                            
                            Node.js - Backend runtime environment for server development
                        

                    

                


                
                

                    

                        
                        AI Models Finetuned
                    

                    

                        

                            
                            BERT - Fine-tuned for legal text understanding and entity extraction
                        

                        

                            
                            GPT-3.5 - Customized for FIR generation and legal document formatting
                        

                        

                            
                            Whisper - Speech-to-text model for audio complaint processing
                        

                        

                            
                            Custom NER Model - Named Entity Recognition for legal document parsing
